0. Запишете си присъствията с парола: 
1. Направете fork на моето github repository.
2. Клонирайте кода от вашият fork на моето github repository.
git clone линк_към_вашето_репозитори

3. Инсталирайте зависимостите.
(трябва да сте влезли в директорията на играта)
npm install
4. Стартирайте сървъра
node server.js
(трябва да имате инсталиран nodemon)
nodemon server.js
npm start

5. Посетете localhost:3000
Ще видите, че играта много време зарежда. Има бъг някъде в кода. Открийте бъга и го оправете.
